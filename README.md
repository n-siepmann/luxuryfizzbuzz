# Luxury Fizzbuzz
A customisable take on a classic coding exercise - enter up to 10 of the following rules:

- Divisible by x
- Square
- Cube
- Prime
- Triangular
- Sum of two cubes
- Power of two
- Mersenne prime
- Evil (ie the number contains an even number of 1s when rendered in binary)

## Stack
Spring Boot, Thymeleaf, Java 8 Predicates
